    **optimizer/gaussian_process.py** (per D-01, Research Pattern 1):
    - `from __future__ import annotations`
    - Import numpy, `scipy.linalg.cho_factor`, `scipy.linalg.cho_solve`
    - `class MinimalGP`: Gaussian Process with RBF kernel for BO surrogate. Follows Rasmussen & Williams 2006.
      - `__init__(self, length_scale: float = 1.0, noise: float = 1e-6)` stores hyperparams. `_X`, `_y`, `_L`, `_alpha` all initialized to None.
      - `fit(self, X: np.ndarray, y: np.ndarray) -> MinimalGP`: Creates a NEW MinimalGP instance (immutable pattern). Computes K = rbf_kernel(X, X) + noise * I. Cholesky factor via cho_factor(K, lower=True). alpha = cho_solve(L, y). Returns new instance with _X, _y, _L, _alpha set.
      - `predict(self, X_new: np.ndarray) -> tuple[np.ndarray, np.ndarray]`: Returns (mean, variance). K_star = rbf_kernel(_X, X_new). mu = K_star.T @ _alpha. v = cho_solve(L, K_star). var = diag(rbf_kernel(X_new, X_new) - K_star.T @ v).clip(min=0.0). Returns (mu, var).
      - `_rbf_kernel(self, X1: np.ndarray, X2: np.ndarray) -> np.ndarray`: Computes exp(-0.5 * ||x1-x2||^2 / length_scale^2). Handle 1D and ND input via np.atleast_2d.
    - All parameters normalized to [0,1] before fitting (normalization done outside GP by caller).

    **optimizer/acquisition.py** (per D-02, Research Pattern 4):
    - `from __future__ import annotations`
    - Import numpy, `scipy.stats.norm`, `scipy.optimize.minimize` as `scipy_minimize`
    - `CONVERGENCE_THRESHOLD: float = 0.01` (1% improvement)
    - `HARD_CAP_EVALUATIONS: int = 200`
    - `def expected_improvement(mean: np.ndarray, variance: np.ndarray, best_observed: float, xi: float = 0.01) -> np.ndarray`: Standard EI formula. z = (best_observed - mean - xi) / sqrt(variance + 1e-12). EI = (best_observed - mean - xi) * norm.cdf(z) + sqrt(variance + 1e-12) * norm.pdf(z). Where variance <= 1e-12, EI = 0.0.
    - `def optimize_acquisition(gp: MinimalGP, best_observed: float, bounds: np.ndarray, n_restarts: int = 5) -> np.ndarray`: Uses L-BFGS-B from scipy.optimize.minimize with n_restarts random starting points. Returns the point with highest EI. bounds is shape (D, 2) in [0,1] normalized space.
    - `def should_stop(acq_values: list[float], n_evaluations: int, window: int = 5) -> bool`: Returns True if n_evaluations >= HARD_CAP_EVALUATIONS. Returns True if len(acq_values) >= window+1 and relative improvement of max(recent window) vs max(previous window) < CONVERGENCE_THRESHOLD. Otherwise False.
