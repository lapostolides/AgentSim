    **writer.py**:
    - `from __future__ import annotations` at top
    - Imports: pathlib.Path, os, stat, json, jinja2, structlog
    - From agentsim.reproducibility.models: ReproducibilityPackage
    - `logger = structlog.get_logger()`
    - Load Jinja2 environment: `jinja2.Environment(loader=jinja2.PackageLoader("agentsim.reproducibility", "templates"), autoescape=False)` — add `tojson` filter pointing to `json.dumps`

    Function `write_package(pkg: ReproducibilityPackage, output_dir: Path | str | None = None) -> Path`:
    - If output_dir is None, use `Path("reproducibility_output") / pkg.experiment_id`
    - Create output_dir with `output_dir.mkdir(parents=True, exist_ok=True)`
    - Render each template: `env.get_template("Dockerfile.jinja2").render(**pkg.model_dump(mode="python"))`
      - Pass `pkg.model_dump(mode="python")` so all fields are available as template variables
    - Write Dockerfile to `output_dir / "Dockerfile"` (no extension)
    - Write requirements.txt to `output_dir / "requirements.txt"`
    - Write run.sh to `output_dir / "run.sh"`, then set executable bit: `os.chmod(run_sh_path, os.stat(run_sh_path).st_mode | stat.S_IEXEC | stat.S_IXGRP | stat.S_IXOTH)`
    - Write report.md to `output_dir / "report.md"`
    - Write metadata.json: `(output_dir / "metadata.json").write_text(pkg.to_json())`
    - Log info: "reproducibility_package_written" with experiment_id, output_dir=str(output_dir), files written list
    - Return output_dir as Path

    CRITICAL: Jinja2 PackageLoader requires the templates directory to be inside the installed package. Use `jinja2.FileSystemLoader` pointing to `Path(__file__).parent / "templates"` instead — this works for both installed and editable installs:
    ```python
    _TEMPLATE_DIR = Path(__file__).parent / "templates"
    _env = jinja2.Environment(
        loader=jinja2.FileSystemLoader(str(_TEMPLATE_DIR)),
        autoescape=False,
        keep_trailing_newline=True,
    )
    _env.filters["tojson"] = lambda v, indent=None: json.dumps(v, default=str, indent=indent)
    ```
