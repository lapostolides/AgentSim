<objective>
Build the reproducibility data models and environment fingerprint capture system. This plan defines the frozen Pydantic schema for a ReproducibilityPackage, implements deterministic environment capture (git hash, Python version, pinned dependencies, platform), and assembles the package from a completed ExperimentState.

Purpose: The ReproducibilityPackage is the core data structure Phase 4 revolves around. It captures everything needed to re-run an experiment identically: what code was running, what packages were installed, what parameters were used, and what physics validations were applied. Plan 02 builds the Jinja2 report templates and runner integration on top of this foundation.
Output: A new src/agentsim/reproducibility/ package with models, fingerprint capture, and package assembler — plus unit tests.
</objective>

    **fingerprint.py** — deterministic environment capture (no LLM calls):
    - `from __future__ import annotations` at top
    - Imports: sys, platform, subprocess, importlib.metadata, structlog
    - From agentsim.reproducibility.models: EnvironmentFingerprint, PackageVersion
    - `logger = structlog.get_logger()`

    Function `get_git_hash() -> str`:
    - Run `subprocess.run(["git", "rev-parse", "HEAD"], capture_output=True, text=True, timeout=5)`
    - Return stripped stdout on returncode==0, else return ""
    - Never raise — wrap in try/except, log debug on failure

    Function `get_git_branch() -> str`:
    - Run `subprocess.run(["git", "rev-parse", "--abbrev-ref", "HEAD"], ...)`
    - Same safe pattern as get_git_hash()

    Function `is_git_dirty() -> bool`:
    - Run `subprocess.run(["git", "status", "--porcelain"], ...)`
    - Return bool(stdout.strip()) on success, else False

    Function `get_pinned_dependencies() -> list[PackageVersion]`:
    - Use `importlib.metadata.packages_distributions()` to get installed package names
    - For each package name, call `importlib.metadata.version(name)` in try/except
    - Return sorted list of PackageVersion(name=name, version=version)
    - Filter out packages that raise PackageNotFoundError silently
    - Log count at debug level

    Function `capture_fingerprint() -> EnvironmentFingerprint`:
    - Gather: python_version from `sys.version.split()[0]`, platform from `sys.platform`, platform_detail from `platform.platform()`
    - Call get_git_hash(), get_git_branch(), is_git_dirty()
    - Call get_pinned_dependencies() -> packages tuple
    - Return EnvironmentFingerprint(...)
    - Log info: "environment_fingerprint_captured" with python_version, git_hash, package_count

    **package.py** — assembles ReproducibilityPackage from ExperimentState:
    - `from __future__ import annotations` at top
    - Imports: datetime, Any, structlog
    - From agentsim.state.models: ExperimentState
    - From agentsim.reproducibility.fingerprint: capture_fingerprint
    - From agentsim.reproducibility.models: ReproducibilityPackage
    - `logger = structlog.get_logger()`

    Function `_extract_parameter_history(state: ExperimentState) -> tuple[dict[str, Any], ...]`:
    - Build one dict per scene in state.scenes: {"scene_id": s.id, "parameters": s.parameters, "iteration": state.iteration}
    - Also include plan parameters if plan exists: {"plan_id": state.plan.id, "metrics": state.plan.metrics} if state.plan else {}
    - Return as tuple of dicts

    Function `_serialize_physics_validations(state: ExperimentState) -> tuple[dict[str, Any], ...]`:
    - For each pv in state.physics_validations: call `pv.model_dump()`
    - Return tuple of those dicts

    Function `generate_package(state: ExperimentState) -> ReproducibilityPackage`:
    - Call capture_fingerprint()
    - Extract parameter_history from _extract_parameter_history(state)
    - Extract physics_validation_results from _serialize_physics_validations(state)

        raw_hypothesis=state.raw_hypothesis,

        parameter_history=parameter_history,
        physics_validation_results=physics_validation_results,
        scene_count=len(state.scenes),
        iteration_count=state.iteration,
        status=state.status.value,
      )
    - Log info: "reproducibility_package_generated" with experiment_id, git_hash, scene_count
