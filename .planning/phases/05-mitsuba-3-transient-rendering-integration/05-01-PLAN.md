    Create validation.py (per D-09, D-10) with these exact functions:

    **validation.py**:
    - `from __future__ import annotations` at top
    - Import: numpy, structlog, pydantic BaseModel
    - `SPEED_OF_LIGHT = 299_792_458.0`
    - `logger = structlog.get_logger()`

    Class `TransientValidationResult(BaseModel, frozen=True)`:
    - measured_peak_ns: float
    - expected_peak_ns: float
    - delta_ns: float
    - tolerance_ns: float
    - match: bool

    Function `opl_to_time_ns(opl_m: float) -> float`:
    - `return (opl_m / SPEED_OF_LIGHT) * 1e9`

    Function `time_ns_to_opl(time_ns: float) -> float`:
    - `return time_ns * 1e-9 * SPEED_OF_LIGHT`

    Function `extract_peak_timing_ns(transient_data: np.ndarray, bin_width_opl: float, start_opl: float = 0.0) -> float`:
    - Sum over spatial and channel dims: `temporal_profile = transient_data.sum(axis=(0, 1, 3))` for (W, H, T, C) shaped data
    - Handle edge cases: if temporal_profile max is 0 or all zeros, log warning and return opl_to_time_ns(start_opl)
    - `peak_bin = int(np.argmax(temporal_profile))`
    - `peak_opl = start_opl + peak_bin * bin_width_opl`
    - `return opl_to_time_ns(peak_opl)`
    - Log: `logger.debug("peak_timing_extracted", peak_bin=peak_bin, peak_opl=peak_opl, peak_ns=result)`

    Function `validate_peak_timing(measured_peak_ns: float, expected_peak_ns: float, tolerance_ns: float = 0.5) -> TransientValidationResult`:
    - `delta_ns = abs(measured_peak_ns - expected_peak_ns)`
    - Create and return `TransientValidationResult(measured_peak_ns=measured_peak_ns, expected_peak_ns=expected_peak_ns, delta_ns=delta_ns, tolerance_ns=tolerance_ns, match=delta_ns <= tolerance_ns)`
    - Log: `logger.info("peak_timing_validated", match=result.match, delta_ns=delta_ns, tolerance_ns=tolerance_ns)`
