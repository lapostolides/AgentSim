Design Annotations
Immutable State
All transitions return new ExperimentState. Pydantic frozen=True on every model. No mutation anywhere.
No eval()
Formula strings parsed via sympy.sympify with restricted namespace. Constants from registry, unit prefix auto-detection.
BFS not DFS
Handles cyclic TF graphs cleanly with visited-edge tracking. Max 100 iterations guard.
Domain-Agnostic
New domains added via YAML directory only. No Python changes needed. Keyword detection auto-loads bundles.
Budget-Free Physics
Propagation and optimization are pure Python. Zero LLM tokens consumed. LLM fallback only for unknown parameters.
Lazy Imports
Physics reasoning imported inside runner loop to avoid circular deps. Bundle loaded once, cached globally.
